# Flex Layouts

There are a number of utility classes available for creating flex layouts. These classes are designed to create dynamic layouts without writing any custom styles. For more advanced layouts, check out Kickstand UI's grid system.

TIP

For additional information on flexbox check out this great guide on CSS Tricks (opens new window). Also check out Flexbox Zombies (opens new window) for a fun way to learn more about using flexbox.

# Responsive

All of the flexbox utility classes have responsive capabilities as well. If you prefix any of these classes with a responsive identifier (xxs-xl), you can modify the layout based on the user's screen size.

Item 1
Item 2
Item 3
<div class="display-flex space-between md:flex-column md:justify-start" style="height: 250px;">
    ...
</div>
